# Contributing

Thanks for contributing.

This repository is designed for learning, so small pull requests are welcome.

## Suggested Workflow

1. Fork the repository.
2. Create a branch:

```bash
git checkout -b add-topic-filter
```

3. Make your change.
4. Run tests:

```bash
pytest
```

5. Commit your work:

```bash
git add .
git commit -m "Add topic filter"
```

6. Open a pull request.

## Code Style

- Keep functions small and readable.
- Add or update tests when behavior changes.
- Prefer clear names over clever code.
- Keep beginner-friendliness in mind.

## Good First Contribution Ideas

- Add a new test
- Improve README examples
- Add a new field to notes
- Add filtering or sorting
- Improve error messages
